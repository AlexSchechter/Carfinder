Car Finder App

This app let's the user search a car database on the server via an internal API. The drop down menues are populated dynamically based on the user's choices.

Once a car has been chosen, the user can see a picture of that car. This is achieved by an image search using a BING API.

The user can also check for details of any recalls in the USA for the chosen car. This is done via a API to the NHTSA database.
