Car Finder App

This app is an excercise in using internal and external APIs.

let's the user search a car database of over 60,000 cars on the server via an internal API. The drop down menus are populated dynamically based on the user's choices.

Once a car has been chosen information available in the local database is displayed to the other. This information includes for example engine size, doors, weight, length, fuel type, etc... However, please note that the database is missing some of the information for some cars and in those cases only the available information will be displayed.

Using a BING API 5 pictures for the car will also be displayed.

The user can also check for details of any recalls in the USA for the chosen car. This is done via a API to the NHTSA database.

The back-end has been programmed with C# and the front-end with AngularJS.
